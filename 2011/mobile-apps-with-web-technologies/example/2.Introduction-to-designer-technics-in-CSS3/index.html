About

A lot of the standards talked about and displayed in this document are still under development. You would have to use vendor prefix for some of the CSS declarations.

A vendor prefix is a browsers implantation of a standard or a work in progress standard.

					
-webkit-something /* Webkit based browsers, Safari, Chrome, Safari on iOS, some of the new Blackberry stuff ++ */
-moz-something /* Mozilla base brwosers, Firefox ++ */
-o-something /* Opera */
-khtml-something (not commonly used, cater for a very tiny percentage of the market)
-ms-something /*Internet Explirer*/
something /* Always end with the standard decleration without any prefix */
